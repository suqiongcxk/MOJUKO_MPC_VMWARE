### 改动 1：URDF → MuJoCo MJCF 转换

- 使用 MuJoCo `compile` 工具：`compile Creeper.urdf creeper.xml`
- 自动生成：运动学树、惯量参数、碰撞几何体（STL 网格 → 圆柱/盒子/球体原语）
- 手动补充：
  - base body + floating joint（free joint，6 DOF）
  - 地面（floor plane）+ 光照（2 个 directional light）+ 天空盒
  - IMU sensor site（imu_sensor_site）
  - 12 个 jointpos + 12 个 jointvel 传感器（关节编码器）
  - base_quat / base_pose / base_angvel / base_linvel 传感器
  - 12 个 motor actuator（力矩上限 23.7 Nm）
  - 碰撞分组（group 1 = 视觉，group 2 = 碰撞）

### 改动 2：关节命名对齐

- 原因：`ModelSettings.h` 硬编码了关节白名单 `{"LF_HAA", "LF_HFE", "LF_KFE", "RF_HAA", "RF_HFE", "RF_KFE", "LH_HAA", "LH_HFE", "LH_KFE", "RH_HAA", "RH_HFE", "RH_KFE"}`
- Pinocchio 通过 `createPinocchioInterface(urdf, jointNames)` 加载时，不在白名单中的关节会被删除 → 导致 numJointState=0
- 修改范围：robot.urdf + creeper.xml

| 原名 | 改为 |
|------|------|
| FL_hip_joint | LF_HAA |
| FL_thigh_joint | LF_HFE |
| FL_calf_joint | LF_KFE |
| FR_hip_joint | RF_HAA |
| FR_thigh_joint | RF_HFE |
| FR_calf_joint | RF_KFE |
| RL_hip_joint | LH_HAA |
| RL_thigh_joint | LH_HFE |
| RL_calf_joint | LH_KFE |
| RR_hip_joint | RH_HAA |
| RR_thigh_joint | RH_HFE |
| RR_calf_joint | RH_KFE |

- actuator 和 sensor 中的 joint 引用同步更新

### 改动 3：足端命名对齐

- 原因 1：`MujocoSimulation.cpp:96` 硬编码 foot geom 名 `{"LF_FOOT_geom", "RF_FOOT_geom", "LH_FOOT_geom", "RH_FOOT_geom"}`
- 原因 2：`ModelSettings.h` 硬编码 contact body 名 `{"LF_FOOT", "RF_FOOT", "LH_FOOT", "RH_FOOT"}`
- 修改范围：robot.urdf + creeper.xml + task.info（碰撞检测对）

| 原名 | 改为 |
|------|------|
| FL_foot (link) + FL_foot_joint | LF_FOOT + LF_foot_fixed |
| FR_foot (link) + FR_foot_joint | RF_FOOT + RF_foot_fixed |

| RR_foot (link) + RR_foot_joint | RH_FOOT + RH_foot_fixed |

- creeper.xml 中每个 foot 从 calf 内的一个 geom 改为独立的 body（含 mass 0.01kg + visual geom + collision geom）
